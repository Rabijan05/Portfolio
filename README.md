## ğŸ“ Customization

### Update Personal Information
- **Name & Bio**: Edit text in `index.html`, `about.html`, `contact.html`
- **Contact Details**: Update email, phone, and social links in `contact.html`
- **Skills**: Modify skill tags in `about.html`
- **Projects**: Update project descriptions and links in `projects.html`

### Styling
- **Colors**: Adjust CSS variables in `styles.css` (`:root` section)
- **Fonts**: Modify font-family and font-sizes in `styles.css`
- **Animations**: Adjust animation timings in `script.js` and `styles.css`

### Color Scheme
```css
--primary-color: #00d4ff;      /* Cyan accent */
--secondary-color: #ff6b9d;    /* Pink accent */
--text-dark: #ffffff;           /* Light text */
--text-light: #b0b0b0;          /* Gray text */
--bg-light: #1a1a1a;            /* Light dark bg */
--bg-white: #0d0d0d;            /* Main dark bg */
```
